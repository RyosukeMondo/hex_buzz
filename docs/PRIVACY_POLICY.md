# Privacy Policy for HexBuzz

**Last Updated**: January 17, 2026

**Effective Date**: January 17, 2026

## Introduction

HexBuzz ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our HexBuzz application (the "App"), available on Windows, Android, iOS, and Web platforms.

Please read this Privacy Policy carefully. By using the App, you agree to the collection and use of information in accordance with this policy.

## Information We Collect

### 1. Personal Information

When you sign in with Google, we collect:

- **Google Account Information**:
  - Email address
  - Display name
  - Profile photo URL
  - Google User ID

We only collect this information when you explicitly choose to sign in. Anonymous gameplay is supported without signing in.

### 2. Game Data

We automatically collect:

- **Gameplay Information**:
  - Completed levels and star ratings
  - Completion times
  - Daily challenge results
  - Progress and achievements
  - Game preferences and settings

### 3. Technical Information

We collect:

- **Device Information**:
  - Device type and model
  - Operating system and version
  - App version
  - Device language settings

- **Usage Analytics** (Optional):
  - App launch and session duration
  - Feature usage statistics
  - Error logs and crash reports
  - Performance metrics

### 4. Notification Data

If you enable push notifications:

- **Device Token**: Unique identifier for push notification delivery
- **Notification Preferences**: Which types of notifications you've enabled

## How We Use Your Information

We use the collected information for:

### 1. Core Functionality
- Saving your game progress across devices
- Enabling cloud save and sync features
- Authenticating your account
- Displaying your profile in leaderboards

### 2. Competitive Features
- Ranking you on global leaderboards
- Processing daily challenge submissions
- Comparing your scores with other players
- Sending notifications for daily challenges and rank changes

### 3. App Improvement
- Analyzing gameplay patterns to improve level design
- Identifying and fixing bugs and crashes
- Optimizing app performance
- Understanding feature usage

### 4. Communication
- Sending push notifications (if enabled):
  - New daily challenges
  - Significant rank changes
  - Re-engagement reminders
- Responding to support requests

## Data Storage and Security

### 1. Where We Store Data

Your data is stored using Google Cloud Platform services:

- **Firebase Authentication**: Secure Google Sign-In credentials
- **Cloud Firestore**: Game progress, scores, and leaderboard data
- **Firebase Cloud Messaging**: Push notification tokens

**Data Location**: United States and global Google Cloud data centers

### 2. Security Measures

We implement industry-standard security measures:

- Encrypted data transmission (HTTPS/TLS)
- Firebase security rules to protect user data
- Secure authentication via Google OAuth 2.0
- Regular security updates and patches
- No storage of sensitive payment information (app is free)

### 3. Data Retention

- **Active Accounts**: Data retained while account is active
- **Inactive Accounts**: Data retained for 3 years of inactivity
- **Deleted Accounts**: Data permanently deleted within 30 days

## Your Privacy Rights

### 1. Access and Control

You have the right to:

- **Access**: View all data we have about you
- **Correction**: Update your profile information
- **Deletion**: Request complete data deletion
- **Portability**: Export your game data
- **Opt-Out**: Disable analytics and notifications

### 2. How to Exercise Rights

**View Your Data**:
- Open App → Settings → Account → View Data

**Update Profile**:
- Your Google profile information updates automatically
- Game settings can be changed in-app

**Delete Your Data**:
- Email: support@hexbuzz.com
- Subject: "Data Deletion Request"
- Include: Your registered email address
- We'll process within 30 days

**Opt-Out of Analytics**:
- Settings → Privacy → Disable Analytics

**Disable Notifications**:
- Settings → Notifications → Toggle off desired types

### 3. GDPR Rights (EU Users)

If you're in the European Economic Area (EEA), you have additional rights:

- Right to erasure ("right to be forgotten")
- Right to restrict processing
- Right to data portability
- Right to object to processing
- Right to withdraw consent

To exercise these rights, contact: support@hexbuzz.com

## Children's Privacy
