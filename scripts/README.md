# Jules CI Scripts

Scripts for building this Flutter project in Jules CI environment.

## Setup Scripts

### `flutter_setup.sh`
Modular Flutter/FVM setup script for CI/CD environments.

**Prerequisites:**
- Android SDK installed and `ANDROID_HOME` environment variable set
- Git repository initialized

**Usage:**
```bash
bash scripts/flutter_setup.sh
```

### `jules_setup.sh`
Complete Jules CI setup script.

**What it does:**
1. Installs system dependencies
2. Downloads and configures Android SDK
3. Runs Flutter/FVM setup


**Usage in Jules CI:**
```bash
bash scripts/jules_setup.sh
```

## Build Scripts

**Usage:**

bash scripts/build_android.sh

**Output:** \`build/app/outputs/flutter-apk/app-release.apk\`
